About MateMonkey

Why?

MateMonkey tries to be a worthy successor to the matekarte which was orginally made by Pascal Jungblut and taken offline. The original dataset of the matekarte has been imported into MateMonkey. Using MateMonkey, Mate drinkers can find and add dealers which sell their favorite drink.

What?

Using MateMonkey you can not only find a dealer near you that sells a special brand of Mate, you can even filter them by type.
You are in charge of choosing the location for tonight? Great! Why not go to a club, restaurant or bar that sells Mate (or even Tschunk)!

Some dealers are quite expensive even tough a cheaper dealer might just be around the corner.
With MateMonkey you can lookup (and update) the current inventory and prices for each dealer.

How?

Will MateMonkey be opensource?
I haven't decided yet whether the project as a whole will be opensource. I plan to make the frontend opensource and put it on GitHub once I am statisfied with the sourcecode.

Is there a (public) API?
Well, since this the frontend is written using pure AngularJS a Padrino JSON Backend handles all the data. You can find documentation for the API on GitHub. There is also a dedicated test platform for you to test your scripts/apps running at http://playground.matemonkey.com. The API can be considered final, however some details might still change in the future (see the documentation for more information).

Is there a test API so I can test my ruby script that runs on my IoT refrigerator?
Yes!. (see above)
